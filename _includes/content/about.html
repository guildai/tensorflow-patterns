TensorFlow Patterns is a collection of patterns in the spirit of Christopher Alexanders Timeless Way of Building. The project seeks to catalog developer experience and distill patterns that can improve our use of TensorFlow.

Start by checking out the patterns. If you see some that you like, try them and see what you think!

This is a 100% open community project that encourages contributions from any and all. If you'd like to contribute please check out our super straight forward contribution policy.
